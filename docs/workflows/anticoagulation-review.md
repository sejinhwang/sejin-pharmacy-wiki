# Anticoagulation Review

## Purpose

Structured pharmacist review of anticoagulation indication, appropriateness, safety, monitoring, and transition-of-care issues.

## Checklist

- Indication and intended duration
- Thrombotic and bleeding risk
- Renal function and dose appropriateness
- Drug interactions
- Recent procedures, bleeding, falls, anaemia, or thrombocytopaenia
- Adherence and administration feasibility
- Monitoring and follow-up plan

## Common agents

- Apixaban / Eliquis
- Rivaroxaban / Xarelto
- Warfarin / Coumadin
- Enoxaparin / Lovenox
- Dalteparin / Fragmin

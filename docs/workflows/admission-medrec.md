# Admission MedRec

## Purpose

Pharmacist admission consult for best possible medication history, admission order reconciliation, and medicines optimisation.

## Core structure

- Reason for consult
- Information sources
- BPMH summary
- Admission orders reviewed
- Discrepancies / medicines-related problems
- Recommendations
- Monitoring and follow-up

## POMR framing

### Initial database

Document source reliability, recent fill history, patient/carer interview, Netcare/PIN, community pharmacy, MAR, and transfer records where applicable.

### Problem list update

Link pharmacist medicines-related problems to the attending provider's active problem list where possible.

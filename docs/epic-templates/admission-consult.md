# Admission Consult Template

## Purpose

Copy-paste-ready pharmacist admission consult template for Epic / Connect Care.

## Core sections

- Reason for consult
- Information sources
- BPMH status
- Admission order reconciliation
- Active medicines-related problems
- Recommendations
- Monitoring / follow-up

## Template development notes

Keep formatting simple for Epic paste behaviour. Prefer clear headings, short bullets, and minimal Markdown complexity.

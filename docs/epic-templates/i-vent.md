# i-Vent Note Bank

## Purpose

Short, structured i-Vent note phrases for pharmacist interventions.

## Categories

- MedRec clarification
- Renal dose adjustment
- Antimicrobial stewardship
- Anticoagulation review
- Therapeutic substitution
- Order clarification
- Discharge preparation

## Style

Use concise UK English, clear action taken, rationale, and expected follow-up.

# Epic SmartLists

## Purpose

Reference page for Epic / Connect Care SmartList-compatible documentation patterns.

## Notes

- Keep output clean for copy/paste.
- Separate selection prompts from final note text where possible.
- Prefer remark-only checklists when SmartList output would clutter the clinical note.

## Common use cases

- Purpose of review
- Administration concerns
- Monitoring requirements
- Pharmacological restraint review
- Medicines review care-plan alignment

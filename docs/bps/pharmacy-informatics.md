# Pharmacy Informatics

Clinical informatics content for medication-use technology, clinical decision support, system build, operations, analytics, and governance.

## Core Domains

| Domain | Examples |
| --- | --- |
| Clinical workflow | Ordering, verification, preparation, dispensing, administration, monitoring, transitions of care. |
| Operational workflow | Technology, system build, implementation, change management, governance. |
| Data and information management | Data models, coding standards, analytics, reporting, and visualisation. |

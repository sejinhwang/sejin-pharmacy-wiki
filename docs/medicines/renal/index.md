# Renal

Use this section for renal medicine review, dose adjustment prompts, nephrotoxin screening, and monitoring templates.

## Build-Out Checklist

| Topic | Notes |
| --- | --- |
| Renal estimate | Document SCr trend, CrCl/eGFR, AKI versus CKD context, dialysis status, and body size considerations. |
| High-risk medicines | Anticoagulants, antimicrobials, diabetes agents, opioids, gabapentinoids, ACEi/ARB, diuretics, NSAIDs, and lithium. |
| Monitoring | Renal function, electrolytes, drug levels where indicated, toxicity symptoms, and reassessment interval. |
| Sick-day review | Medicines to reassess during dehydration, acute illness, AKI, or contrast exposure. |

!!! caution
    Dose adjustment depends on indication, renal trajectory, assay limits, dialysis modality, and local policy.

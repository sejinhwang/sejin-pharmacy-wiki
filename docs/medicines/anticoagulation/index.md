# Anticoagulation

Use this section for anticoagulant selection notes, monitoring prompts, interaction checks, peri-procedural considerations, and documentation templates.

## Build-Out Checklist

| Topic | Notes |
| --- | --- |
| Indication | AF, VTE treatment, VTE prophylaxis, mechanical valve, or other documented indication. |
| Patient factors | Renal function, hepatic function, age, weight, bleeding history, falls risk, adherence, and pregnancy status where applicable. |
| Monitoring | Baseline and follow-up labs, bleeding assessment, thrombotic symptoms, and dose-change triggers. |
| Interactions | Antiplatelets, NSAIDs, strong CYP/P-gp modulators, herbals, and alcohol use. |

!!! caution
    Verify dosing, duration, and monitoring against current local guidance and product monographs before use.

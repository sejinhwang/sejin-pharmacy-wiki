# Diabetes

Use this section for diabetes medicines review, hypoglycaemia risk assessment, renal considerations, sick-day prompts, and counselling templates.

## Build-Out Checklist

| Topic | Notes |
| --- | --- |
| Glycaemic context | A1C goal, recent readings, hypoglycaemia history, steroid use, nutrition status, and care setting. |
| Medicine review | Insulin, metformin, SGLT2 inhibitors, GLP-1 receptor agonists, sulfonylureas, and other agents. |
| Safety | Renal function, dehydration risk, infection risk, frailty, falls risk, and hypoglycaemia awareness. |
| Counselling | Monitoring plan, hypoglycaemia management, sick-day plan, and follow-up needs. |

!!! caution
    Individualise glycaemic targets and deprescribing decisions using local policy and patient-specific risk.

# Pain

Use this section for analgesic review, opioid stewardship, adjuvant therapy notes, renal considerations, and counselling templates.

## Build-Out Checklist

| Topic | Notes |
| --- | --- |
| Pain assessment | Location, type, severity, function, duration, goals, and response to current therapy. |
| Medicine review | Non-opioids, opioids, adjuvants, topical options, bowel regimen, and naloxone where appropriate. |
| Safety | Sedation, falls, constipation, renal function, hepatic function, respiratory risk, and interactions. |
| Follow-up | Functional outcomes, adverse effects, taper plan, monitoring frequency, and escalation criteria. |

!!! caution
    Confirm opioid conversions, renal dosing, and taper plans with authoritative references before use.

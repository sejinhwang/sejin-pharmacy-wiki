# Antimicrobials

Use this section for antimicrobial review notes, stewardship prompts, renal adjustment references, monitoring, and counselling points.

## Build-Out Checklist

| Topic | Notes |
| --- | --- |
| Syndrome | Document suspected or confirmed infection source and severity. |
| Microbiology | Culture results, susceptibilities, prior colonisation, and recent antimicrobial exposure. |
| Stewardship | Indication, empiric rationale, de-escalation plan, duration, IV-to-PO suitability, and stop/review date. |
| Safety | Allergies, renal function, QT risk, C. difficile risk, cytopaenia risk, and drug interactions. |

!!! caution
    Prefer local antibiograms, Bugs & Drugs, AHS guidance, and current stewardship recommendations.

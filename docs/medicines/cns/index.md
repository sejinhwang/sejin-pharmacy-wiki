# CNS

Use this section for CNS medicines review, psychotropic monitoring, anticholinergic burden, falls risk, and tapering documentation.

## Build-Out Checklist

| Topic | Notes |
| --- | --- |
| Indication | Document target symptoms, diagnosis, duration, and response. |
| Safety | Sedation, cognition, falls, QT risk, anticholinergic burden, EPS, serotonin toxicity risk, and withdrawal risk. |
| Monitoring | Mental status, sleep, behaviour, mood, movement symptoms, labs where indicated, and functional outcomes. |
| Deprescribing | Current benefit, harms, taper feasibility, monitoring plan, and patient/caregiver goals. |

!!! caution
    Use trauma-informed, patient-centred language and avoid unsupported assumptions about behaviour or diagnosis.

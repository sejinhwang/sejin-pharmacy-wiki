## Import Rules

- Each symptom gets its own local Markdown page.
- The page must contain triage content, differentials, red flags, and medication/practice notes when present.
- Related conditions are written as local text first. Links are added only when a local target exists.
- Notion page links are not used as the reader-facing navigation model.

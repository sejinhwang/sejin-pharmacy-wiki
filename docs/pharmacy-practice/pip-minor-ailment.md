# PIP | Pharmacist Independent Prescribing for Minor Ailment

Status: Local migration map. The previous external-only Notion inventory has been replaced with local target slugs so this page can function as a standalone migration control page.

## Purpose

This section organises pharmacist independent prescribing / minor ailment topics into local wiki targets.

Each condition should eventually become a standalone local page with:

- scope and eligibility
- red flags and referral criteria
- focused assessment
- treatment options
- counselling
- safety-netting
- documentation template
- local policy / formulary review status

## Database Schema

| Property | Purpose |
| --- | --- |
| Condition | Minor ailment or prescribing service page title. |
| System | Body system/category. |
| Scope | Brief condition scope or definition. |
| Provincial Status | Province or programme status, including SK, MB, NS, NB, PEI, QC, ON proposed, and AB APA. |
| Symptoms | Linked symptom sorter entries. |
| Meta-Database Topics | Primary linked meta topics. |
| Related Meta Topics | Secondary linked meta topics. |

## Local Page Creation Checklist

For each condition page:

1. Create the local file at the target slug.
2. Migrate the full body content from the source page.
3. Remove external-only Notion dependency.
4. Add local links to symptom sorter and meta-database pages where available.
5. Mark clinical/local policy review status.
6. Enable the link in this index only after the target file exists.

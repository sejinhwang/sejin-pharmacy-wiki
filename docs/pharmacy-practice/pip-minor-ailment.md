# PIP | Pharmacist Independent Prescribing for Minor Ailment

Status: Public migration index. Direct Notion condition links have been replaced with local wiki targets.

## Purpose

This section organises pharmacist independent prescribing / minor ailment topics into local public wiki pages.

Each condition page should eventually include:

- scope and eligibility
- red flags and referral criteria
- focused assessment
- management principles
- counselling
- safety-netting
- documentation template
- local policy / formulary review status

Public pages must not include patient identifiers, personal prescriber identifiers, registration numbers, signatures, or private workflow credentials.

## Status Legend

| Status | Meaning |

| Under construction | Local placeholder exists but full content is not yet migrated. |
| Blocked / manual creation needed | Connector could not create the file; create manually before migration. |

## Local Page Publication Checklist

Before moving a page from draft to publish-ready:

1. Full public-safe body content is present locally.
2. Notion dependency is removed from the reader experience.
3. Local links to symptom sorter and meta-database pages are added where available.
4. No patient identifiers, personal prescriber identifiers, registration numbers, signatures, or private workflow credentials are present.
5. Dose- or product-specific tables are either omitted or reviewed against local policy and current references.
6. Clinical/local policy review status is clearly documented.
7. The page no longer contains an `Under construction` warning.

## Link and Anchor Verification

- Direct Notion condition links have been removed from the main inventory.
- Public-safe draft pages and existing placeholders are linked.
- Blocked or missing local files remain unlinked as code-formatted target slugs.
- Existing local Symptom Sorter support links are enabled where relevant.
- Table text avoids unescaped pipe characters so MkDocs renders rows correctly.

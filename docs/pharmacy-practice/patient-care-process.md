# Patient Care Process

Structured pharmacy care process content.

## Core Flow

| Step | Purpose |
| --- | --- |
| Collect | Gather patient context, medicine history, symptoms, labs, and barriers. |
| Assess | Identify medicine related problems, risks, and care priorities. |
| Plan | Set goals, choose interventions, and define monitoring. |
| Implement | Counsel, prescribe or recommend, communicate, and document. |
| Follow-up | Monitor outcomes, adverse effects, adherence, and escalation needs. |

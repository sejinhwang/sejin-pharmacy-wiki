# Renal Dose Adjustment

## Purpose

Clinical guide for reviewing medicine appropriateness in CKD, AKI, and fluctuating renal function.

## Core checks

- Confirm current Cr/eGFR and baseline renal function
- Assess trajectory rather than single value alone
- Review indication and therapeutic alternatives
- Adjust dose or interval where required
- Consider temporary holds in AKI
- Document monitoring and restart criteria

## High-yield groups

- Antimicrobials
- Anticoagulants
- Diabetes medicines
- Gabapentinoids
- Opioids
- Cardiovascular medicines

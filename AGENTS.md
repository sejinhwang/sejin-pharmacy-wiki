# AGENTS.md

## Project

This repository is a clinical pharmacy wiki for medicines optimisation, Epic / Connect Care documentation templates, POMR workflows, and hospital/LTC pharmacy practice.

## Language and terminology

- Use UK English spelling and grammar.
- Use British/Canadian clinical terminology where appropriate.
- Prefer paracetamol, A&E, medicines, FBC, WCC, Hb, Plt, cytopaenia, neutropaenia, thrombocytopaenia, pancytopaenia, haematology, haemoglobin, haematuria, and haemodynamically.
- Use medicine frequency abbreviations OD, BD, TDS, and QDS.
- Use nocte rather than HS.
- Use mane for morning dosing where personalised dosing directions are required.

## Clinical safety

- Do not invent clinical recommendations.
- Flag uncertainty clearly.
- Prefer authoritative sources, including local AHS / Connect Care guidance, Health Canada, CPS / RxTx, product monographs, Bugs & Drugs, Choosing Wisely Canada, and recognised guidelines.
- For high-risk topics, include monitoring, renal adjustment, contraindications, and escalation criteria.
- This wiki is decision-support and documentation support only, not a replacement for clinical judgement.

## Documentation style

- Use structured headings.
- Prefer concise tables for dosing, monitoring, renal adjustment, and counselling points.
- For Epic copy-paste templates, avoid complex Markdown that may paste poorly.
- Keep clinical notes copy-paste-ready.
- Use POMR structure where applicable: initial database, problem list, initial plan, progress note, and problem list update.

## Repository conventions

- All wiki pages live under docs/.
- Reusable templates live under templates/.
- Data tables live under data/.
- Scripts live under scripts/.
- Do not commit PDFs, PHI, patient identifiers, API keys, local exports, or secrets.
